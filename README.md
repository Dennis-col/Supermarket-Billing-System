# Supermarket Billing System

## Project Description
The Supermarket Billing System is an open-source console application written in Python. It is designed to modernize and automate the customer checkout experience for retail entities like SmartBuy Supermarket. The software cleanly transitions manual ledger and calculator computing methods into an error-free, streamlined computational workflow.

## How the Program Works
1. Continuous Operations: The software operates within an interactive processing environment, running through successive customer iterations until explicitly instructed to terminate by the cashier.
2. Item Registration: Cashiers iteratively input item specifications (Product name, individual unit cost, and volume purchased).
3. Array Caching: Information sets are stored safely across localized index arrays.
4. Discount Rule Assessment: Upon structural termination of an item set entry, data-structures verify whether aggregated costs cross the threshold of Le 500. If parameters are met, a precise 10% markdown calculation activates automatically.
5. Receipt Rendering: The platform structures, computes, and aligns dynamic content tables, presenting a clean terminal receipt.

## How to Run the Program
Ensure you have Python 3.x installed on your target machine operating system.
1. Clone this repository locally or download the source files.
2. Open your system Terminal or Command Prompt.
3. Target the execution workspace folder: cd Supermarket-Billing-System
4. Execute the Python initialization call: SBS.py
